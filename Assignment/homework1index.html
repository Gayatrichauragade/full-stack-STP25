What is HTML

HTML, short for HyperText Markup Language, is the foundation of the World Wide Web. It's a markup language used to structure and design web pages, defining how text, images, and multimedia content are displayed in a web browser.

HTML tags are broadly categorized into paired (container) tags and unpaired (empty) tags
